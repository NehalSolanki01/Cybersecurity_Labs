# 🔐 Cybersecurity Labs Portfolio

Welcome to my hands-on cybersecurity lab repository. This project documents my journey from foundational concepts to advanced real-world security scenarios through practical labs.

---

## 🎯 Objective

The goal of this repository is to:

* Build a strong foundation in cybersecurity concepts
* Develop a structured problem-solving mindset
* Gain hands-on experience with industry tools
* Simulate real-world attacks and defenses
* Progress from beginner to advanced (SOC / Network Security / Blue Team)

---

## 🧠 Learning Approach

Instead of memorizing commands, I focus on understanding:

1. **What is happening?** (Attack / behavior)
2. **How can it be detected?** (Logs / monitoring)
3. **How can it be prevented?** (Security controls)

Each lab follows this workflow:

```text
Attack → Detection → Analysis → Response → Prevention
```

---

## 🧰 Tools & Technologies

This repository will cover a wide range of tools to gain practical exposure:

* Kali Linux (Attacker machine)
* Ubuntu Server (Target machine)
* Hydra (Brute force attacks)
* Nmap (Port scanning)
* Wireshark (Packet analysis)
* tcpdump (Traffic capture)
* Fail2Ban (Intrusion prevention)
* UFW Firewall (Access control)
* Splunk / ELK (SIEM – upcoming)
* Snort / Suricata (IDS/IPS – upcoming)

---

## 🏗️ Lab Environment

All labs are performed in a controlled virtual environment using:

* VMware Workstation
* Multiple virtual machines (Kali Linux + Ubuntu Server)
* Isolated network (NAT)

This ensures safe simulation of attacks without impacting real systems.
